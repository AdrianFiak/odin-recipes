Shake de banane cu mere

Se curata o banana si un mar si se taie cubulete. Se mai adauga o lingura de afine si una de zmeura.

Se mai pune si o lingurita de seminte de chia dupa care se pune 150 ml de lapte de orez.

Se lasa se se inmoaie semintele de chia dupa care se mixeaza si se serveste.

Ingrediente

  • o banana
  • un mar
  • o lingura de afine
  • o lingura de zmeura
  • o lingurita de seminte de chia
  • 150 ml de lapte de orex
